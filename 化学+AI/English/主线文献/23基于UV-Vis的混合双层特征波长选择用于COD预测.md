# 核心创新点总结

## 1. 双层特征波长选择方法
本文创新性地提出了一种结合包装（AFSA）和嵌入（EN）方法的双层特征波长选择模型。AFSA首先进行全局优化选出初步特征波长，然后通过EN进一步精细选择，克服了传统方法在高维数据中的冗余和共线性问题，从而提高了特征选择的效率和精度。

## 2. AFSA-EN组合模型
将人工鱼群算法（AFSA）和弹性网（EN）回归算法结合，形成了一个具有全局搜索和正则化能力的双层特征选择模型。该组合方法在提取有效特征波长时，能够优化模型性能，并有效应对数据中的共线性和冗余问题。



# 化学需氧量(COD)预测的混合双层特征波长选择方法


## 2. 方法
### 2.1 特征波长选择模型
UV-Vis吸收光谱数据中，波长点之间具有高共线性和冗余性，因此直接处理所有波长作为特征会导致模型过于复杂并浪费计算资源。本文使用AFSA算法在所有波长范围内进行初步特征选择，然后使用EN回归算法对其进行优化，最终获得最优的特征波长。

### 2.2 浑浊度补偿模型
本文基于朗伯比尔定律，通过减去水中浑浊颗粒对吸光度的影响来补偿浑浊度的影响。浑浊度补偿后，经过补偿的光谱数据被输入到MLP中，进一步提高COD的预测精度。

### 2.3 COD测量模型
为了提高COD预测的精度，选取220-450nm范围内的波长作为有效波长，将补偿后的光谱数据输入到MLP中进行COD回归预测。MLP作为一种非线性模型，能够较好地捕捉复杂的非线性关系。

## 3. 实验结果
### 3.1 特征波长选择结果
通过与GA-EN、EN和PLS等方法的比较，AFSA-EN方法在特征波长选择方面表现最佳。AFSA-EN方法能够较好地处理波长数据中的共线性问题，提高了模型的预测精度。

### 3.2 COD预测结果
AFSA-EN-MLP模型在所有评估指标上优于其他模型，R²值达到了0.9172，相比VRBF模型，RMSE和MAE分别降低了17.07%和51.12%。这表明AFSA-EN-MLP模型不仅提高了COD预测的准确性，还能有效补偿浑浊度的影响。

## 4. 结论
本文提出了一种基于UV-Vis光谱的COD检测新方法，结合了人工鱼群算法（AFSA）和弹性网（EN）回归算法进行特征波长选择，成功实现了浑浊度补偿，并通过多层感知机（MLP）提高了COD预测的精度。
