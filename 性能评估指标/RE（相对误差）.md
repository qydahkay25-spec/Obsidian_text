1. **核心定义与应用场景**
    - **核心定义**：相对误差是指预测值与真实值之间的误差与真实值的比值，用于衡量预测值偏离真实值的相对程度。其计算公式为：$RE = \frac{|预测值 - 真实值|}{真实值} \times 100\%$。
    - **应用场景**：在各种预测和估算领域广泛应用，如科学实验数据的误差分析、经济预测模型评估、工程测量精度评估等，可直观反映预测结果的准确程度。
2. **基本原理与核心思想**
    - **基本原理**：通过计算预测值与真实值差值占真实值的比例，得到一个无量纲的数值，以统一标准衡量不同量级数据预测的准确程度。
    - **核心思想**：相对误差将误差与真实值关联起来，避免了仅看绝对误差时，因真实值量级不同而导致对误差严重程度判断的偏差。例如，绝对误差为 1，对于真实值是 100 和真实值是 1000 的情况，严重性显然不同，相对误差能更好体现这种差异。
3. **关键步骤与执行流程**
    - **步骤**：首先获取预测值和真实值，然后计算两者差值的绝对值，再将该绝对值除以真实值，最后乘以 100% 得到相对误差百分比。
4. **优势与局限性**
    - **优势**：无量纲，便于不同量级数据间误差比较；能直观反映预测值偏离真实值的相对大小，更符合实际意义上对误差的感受。
    - **局限性**：当真实值接近于 0 时，相对误差可能趋于无穷大，此时相对误差失去意义；对于一些不适合用比例衡量误差的场景（如某些物理量的绝对偏差有严格限制场景），相对误差不能完全反映误差的影响。
5. **适用数据类型与场景边界**
    - **适用数据类型**：适用于数值型数据，无论是连续型还是离散型数值。
    - **场景边界**：适用于希望从相对角度评估预测准确性的场景，但真实值接近 0 时需谨慎使用，或者结合绝对误差等其他指标综合评估。

### Python 代码实现

python

```python
def relative_error(actual, predicted):
    # 核心步骤：计算预测值与真实值差值的绝对值，除以真实值，乘以100%
    try:
        re_value = (abs(predicted - actual) / actual) * 100
        return re_value
    except ZeroDivisionError:
        print("真实值为0，相对误差无意义")
        return None

# 输入示例
actual_value = 50
predicted_value = 45

# 调用函数计算相对误差
result = relative_error(actual_value, predicted_value)
if result is not None:
    print(f"相对误差为: {result}%")
```

- **输出示例说明**：如果真实值不为 0，输出相对误差的百分比数值，如上述代码运行结果为 “相对误差为: 10.0%”；若真实值为 0，输出提示信息 “真实值为 0，相对误差无意义” 。

### 注意事项与优化方向

- **注意事项**：使用时需注意真实值不能为 0；在对比不同组数据相对误差时，要确保数据性质和量级具有可比性。
- **优化方向**：可结合绝对误差等其他误差指标，全面评估预测结果。对于真实值接近 0 的情况，可考虑使用其他专门处理小数值误差评估的方法，如平均绝对误差百分比（MAPE）的变体，对接近 0 的真实值进行特殊处理 。
