# RPD（相对分析误差）

RPD 是回归预测模型中评估模型实用性的核心指标，尤其在近红外光谱、化学计量学、农业/环境预测等领域广泛应用。它通过 **"数据离散程度"** 与 **"模型预测误差"** 的比值，判断模型是否具备实际应用价值。

---

## **一、核心定义与公式**

**公式**：  
$\text{RPD} = \frac{\text{SD（真实值的标准差）}}{\text{RMSEP（预测集均方根误差）}}$

**其中**：

- **$\text{SD}$**：预测集真实值的样本标准差（反映数据本身的离散程度）  
    $SD = \sqrt{\frac{1}{n-1}\sum_{i=1}^n (y_i - \bar{y})^2}$
    
- **$\text{RMSEP}$**：预测集均方根误差（反映模型预测偏差）  
    $RMSEP = \sqrt{\frac{1}{n}\sum_{i=1}^{n} (y_i - \hat{y}_i)^2}$
    
- **$\bar{y}$**：预测集真实值的平均值
    
- **$n$**：预测集样本数量
    

---

## **二、关键解读**

### **1. 取值与模型实用性（行业通用标准）**

|RPD 范围|模型实用性|应用场景|
|---|---|---|
|**RPD < 1.5**|无实用价值（预测效果极差）|需重新优化模型 / 更换特征|
|**1.5 ≤ RPD < 2**|可用于粗略定性区分|仅做趋势判断，不适合定量预测|
|**2 ≤ RPD < 2.5**|可用于定量预测（精度一般）|非高精度要求的场景|
|**2.5 ≤ RPD < 3**|定量预测精度良好|大部分工业/科研场景|
|**RPD ≥ 3**|定量预测精度优秀|高精度要求的场景（如精准检测）|

### **2. 物理意义**

- **RPD 值越大**，说明"数据本身的波动（SD）"远大于"模型的预测误差（RMSEP）"
    
- 这意味着模型能有效区分不同样本的真实值，预测可靠性高
    
- 相反，**RPD 值越小**，说明预测误差接近甚至超过数据本身的自然变异，模型实用价值低
    

### **3. 优点与注意事项**

- **优点**：结合了数据自身特性，提供了相对评价基准，不同数据集间可比性更强
    
- **注意**：RPD 阈值标准会因领域和具体应用略有差异，建议结合其他指标（如 $R^2$、RMSEP）综合评估
